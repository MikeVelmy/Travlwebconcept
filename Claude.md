# Claude Code Project Rules

## Engineering Principles
- Use clean, modular, and functional React components with TypeScript.
- Strictly adhere to Next.js App Router conventions (e.g., `app/page.tsx`).
- Keep components small and reusable; split layouts into a dedicated `components/` directory.

## UI & Styling Guidelines
- Use Tailwind CSS exclusively for styling. Do not inject inline CSS styles or raw CSS files.
- Ensure all pages are fully responsive across mobile, tablet, and desktop devices.
- Focus on clean typography, smooth hover transitions on clickable elements, and generous element padding.

## Commands for Claude Code Reference
- Install Dependencies: `npm install`
- Start Dev Server: `npm run dev`
- Build Verification: `npm run build`

**Side note. Make it Mobile-first. And Add Images to make the site look real and lively.**
